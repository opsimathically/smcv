# Security policy

SMCV is under active development and is not yet safe for real secrets.

Do not place vulnerability details, live credentials, private keys, recovery
material, or production records in public issues, logs, fixtures, or evidence.
Before public distribution, the release phase will establish a private report
channel and supported-version policy. Until then, retain a minimal synthetic
reproduction locally and notify the repository owner through an already trusted
private channel.

Security findings remain active implementation work until fixed and retested.
